# Python Projects
## Guess the Number Game
<b>DESCRIPTION</b><br>
-----------<br>
A simple command-line game where the player tries to guess a randomly<br>
generated number within a specified range. Test your luck and logic!<br>

<b>HOW TO PLAY</b><br>
-----------<br>
1. The computer will randomly select a number within a given range.<br>
2. You will be prompted to enter your guess.<br>
3. The game will tell you if your guess is too high, too low, or correct.<br>
4. Keep guessing until you find the right number!<br>

<b>REQUIREMENTS</b><br>
-------------<br>
Python 3.x
